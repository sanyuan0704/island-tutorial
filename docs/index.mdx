# 初始化项目

现在让我们从零创建一个新的项目。

<CH.Spotlight>

```bash terminal
mkdir island-app && cd island-app
```

---

**Step 1**

首先，你可以通过以下命令创建一个新目录：

```bash terminal focus(1)

```

---

**Step 2**

执行 `npm init -y` 来初始化一个项目。你可以使用 npm、yarn 或 pnpm 安装 Island.js:

```bash terminal
# npm
npm install islandjs
# yarn
yarn add islandjs
# pnpm
pnpm add islandjs
```

---

**Step 3**

然后通过如下命令创建文件:

```js terminal
mkdir docs && echo '# Hello World' > docs/index.md
```

---

**Step 4**

在`package.json`中加上如下的脚本，你直接粘贴过去:

<CH.Code>

```js terminal
mkdir docs && echo '# Hello World' > docs/index.md
```

---

```json package.json
"scripts": {
  "dev": "island dev docs",
  "build": "island build docs",
  "preview": "island start docs"
},
```

</CH.Code>

---

**Step 5**

启动本地开发服务，这样 Island.js 将在 http://localhost:5173 启动 Dev Server。

```js terminal
npm run dev
```
