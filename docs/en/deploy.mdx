# Deploy

You can deploy the project in a variety of ways. This article mainly introduces Vercel and Netlify.

<CH.Scrollycoding>

## Vercel

1. Push your code to the remote repository(GitHub, GitLab, Bitbucket)。
2. Import your Island.js project repository into [Vercel](https://vercel.com/dashboard)。
3. Configure the build command for the project.
4. Trigger the deployment and get the deployment domain name. Example: https://island-tutorial.vercel.app

<div my="5"></div>

```bash config
Build Command: npm run build
Output Directory: docs/.island/dist
```

---

## Netlify

1. Push your code to the remote repository(GitHub, GitLab, Bitbucket)。
2. Import your Island.js project repository into [Netlify](https://app.netlify.com/)。
3. Configure the build command for the project.
4. Trigger the deployment and get the deployment domain name. Example: https://super-puffpuff-77fc9f.netlify.app
