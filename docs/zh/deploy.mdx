# 项目部署

你可以将用多种方式来对项目进行部署，本文主要介绍 Vercel 和 Netlify 两种方式。

<CH.Scrollycoding>

## Vercel 部署

1. 将你的代码推送到远程仓库（GitHub, GitLab, Bitbucket）。
2. 导入你的 Island.js 项目仓库 到 [Vercel](https://vercel.com/dashboard)。
3. 进行项目的构建命令配置。
4. 触发部署，得到部署域名。示例：https://island-tutorial.vercel.app

<div my="5"></div>

```bash config
Build Command: npm run build
Output Directory: docs/.island/dist
```

---

## Netlify 部署

1. 将你的代码推送到远程仓库（GitHub, GitLab, Bitbucket）。
2. 导入你的 Island.js 项目仓库 到 [Netlify](https://app.netlify.com/)。
3. 进行项目的构建命令配置。
4. 触发部署，得到部署域名。示例：https://super-puffpuff-77fc9f.netlify.app
