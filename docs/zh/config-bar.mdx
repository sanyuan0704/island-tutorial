---

**Step 1**

添加配置文件。文件目录结构:

```bash 目录结构
.
├─ docs
│  ├─ .island
# focus
│  │  └─ config.ts
│  └─ index.md
└─ package.json
```

---

**Step 2**

初始化配置文件内容:

```ts config.ts
import { defineConfig } from "islandjs";

export default defineConfig({
  // 具体内容
});
```

---

**Step 3**

添加导航栏配置:

```ts config.ts
import { defineConfig } from "islandjs";

export default defineConfig({
  // focus(1:8)
  themeConfig: {
    nav: [
      {
        text: "首页",
        link: "/",
      },
    ],
  },
});
```

---

**Step 4**

添加侧边栏配置:

```ts config.ts
import { defineConfig } from "islandjs";

export default defineConfig({
  themeConfig: {
    nav: [
      {
        text: "首页",
        link: "/",
      },
    ],
    sidebar: {
      "/guide/": [
        {
          text: "Guide",
          items: [
            {
              text: "Getting Started",
              link: "/guide/getting-started",
            },
          ],
        },
      ],
    },
  },
});
```

---

**Step 5**

启动本地开发服务，这样 Island.js 将在 http://localhost:5173 启动 Dev Server。

```js terminal
npm run dev
```
